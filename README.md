## Phone fly 팀 프로젝트

## 주제 및 목적

- 주제
   - 휴대폰 온라인 판매 사이트
- 목적
   - 이용자와 관리자의 편의성 제공
- 주요 기능
   - 요금제 비교 : 제품 상세 페이지의 버튼 선택을 통한 실시간 요금제 계산 기능
   - 제품 가격 비교 : 이용자가 원하는 제품을 선택하여 비교 기능
   - 색상 선택 : 제품마다 상이한 종류의 색상을 선택하여 미리보기 기능

## 밴치마킹 사이트

  - 스마트투데이(http://smatoday.co.kr/main/main.html) : 제품 비교기능 
  - 오늘의 폰(https://www.todayphone.net/) : 제품 상세 페이지, 메뉴 구성
  - 애플(https://www.apple.com/kr/iphone/) : 디자인 및 기능
  - 삼성(https://www.samsung.com/sec/smartphones/) : 제품 컬러 선택 기능/ 비교 선택 및  저장 기능

## 요구 사항 분석

- 주요기능
  - 상품 상세보기 옵션 선택이 가능한가
  - 상품 상세보기 옵션 선택시 가격이 자동으로 계산되는가
  - 원하는 상품만 선택하여 비교할 수 있는가

- 사이트 기본기능
  
1.회원
 - 관리 (가입, 탈퇴, 정보수정, 아이디 찾기, 비밀번호 찾기)
 - 회원 개인메뉴 (가입신청 내역, 개인 1:1 문의내역, 개인 후기내역)
 - 온라인 가입신청 (구매)
 - 구매후기

2.비회원
 - 회원가입 (ID중복확인, 주소찾기)
 - 약관

3.공통
 - 제조사별 상품 리스트
 - 전체 상품 리스트 (비교 상품 선택 없이 이동시)
 - 자주하는 질문
 - 게시판 (공지사항, 이벤트)

## 후기

- 후기
  
   - 첫 프로젝트이였던만큼 부족한 지식과 경험으로 인해 애로사항이 있었지만,진행하면서 미숙했던 부분을 다시 복습을 하고 이론을 실제로 적용해보면서 최종 결과물을 잘 마무리 할 수 있었습니다. 기술적으로는 스프링부트 , Mybatis 및 AJax 등 을 적용하면서 개발 속도가 2배 이상 빨라지는 경험을 하였고, 기본적인 웹페이지 구동에 대한 지식뿐만 아니라 학습하는 습관과 문제해결 능력을 키웠고 협업의 기본 과정 등 많은걸 깨닳고 발전이 됬던 좋은 경험의 프로젝트였습니다.
